# Dinner Wheel Spinner 🍽️
![spinner](image.png)

I made this dinner wheel spinner because we always had a hard time deciding where to go for dinner. Now we can just spin and go!

## Features
- Smooth canvas-based spinning with easing animation and realistic momentum.
- Hover highlights on wheel segments for preview.
- Customizable dinners and colors in `data.js`—add your favorites easily.
