# Dinner Wheel Spinner 🍽️

Interactive wheel spinner for randomly picking tonight's dinner! Spin for quick meal decisions from 30+ tasty options like Pizza, Sushi, Hotpot, Pho, and more.[web:88]

## Features
- Smooth canvas-based spinning with easing animation and realistic momentum.
- Hover highlights on wheel segments for preview.
- Customizable dinners in `data.js`—add your favorites easily.
- Responsive design, works great on mobile/desktop.
- Clean separation: external CSS/JS files.
